# Nest E-Commerce Backend
## Description

This is the backend for an e-commerce site. Its resposible for handling all API requests, Authentication and Authorization, Interaction with the database through CRUD operations and report queries. 

## Requirements
1. Nest v9+
2. Node v16+

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
